hexo+github搭建简易博客

hexo+github搭建一个自己的博客，平时可以写点心得以及技术分享，本博客名为 “心灵的孤僻“。

创建github帐号

  1. 在github官网上创建一个自己的帐号，例如 username;
  2. 再创建一个名为username.github.io的repo;
  3. 在自己的电脑上配置git。

配置hexo

本博客采用hexo-theme-variant主题，在原配置基础上做了相应的修改。

  • 下载主题；
    git clone https://github.com/yumemor/hexo-theme-varaint.git themes/variant
  • 修改更目录下的_config.yml(在修改之前最好先备份)
    theme: variant
